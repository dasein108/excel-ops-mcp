# Task 20 — User README and Install Guide

Create a comprehensive `README.md` for users of the Excel MCP and `excel-ops` CLI.

## Scope

- Explain the product.
- Document MCP tools.
- Document CLI commands.
- Lead with `uvx`.
- Explain how to install `uv`.
- Include MCP host configuration.
- Mention the agent-agnostic skill path.

## Acceptance Criteria

- README includes `uvx excel-mcp` as the recommended MCP server command.
- README explains installing `uv`:
  - macOS/Linux shell install.
  - Homebrew.
  - Windows PowerShell.
- README documents both executables:
  - `excel-mcp`
  - `excel-ops`
- README documents all MCP tools and CLI commands.
- README includes safe-write workflow: write, diff, commit to new file.
- README mentions Google Sheets and Numbers are not supported in this phase.
